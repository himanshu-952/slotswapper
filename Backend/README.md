SlotSwapper Backend
Peer-to-peer time slot scheduling app backend (Node.js, Express, MongoDB).

Overview
SlotSwapper lets users swap calendar events with others. Users mark events as "swappable" and can exchange slots upon mutual agreement.

Features
User authentication (JWT)

CRUD for calendar events

Swappable slots marketplace & swap request/approval logic

Mongoose models & REST API

MongoURI , JWT secret key in in .env file which is not pushed to github.

Deployed on Render.
